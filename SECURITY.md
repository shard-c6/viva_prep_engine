# Security Policy

We take security seriously. If you discover a vulnerability in VERA, please report it immediately so we can address it before public disclosure.

## Reporting a Vulnerability

**Please do not open a public GitHub issue for security vulnerabilities.**

Instead, please report security vulnerabilities by emailing the maintainer directly.
We will acknowledge your report, investigate the issue, and work on a fix as quickly as possible.

## Supported Versions

| Version | Supported |
| --- | --- |
| 1.0.x | Yes |
| < 1.0 | No |

## Scope

This security policy applies to all code and templates housed in the VERA repository.
It does not apply to external services (GitHub API, Supabase, Gemini AI Studio) used by the pipeline. Please report vulnerabilities for those services directly to their respective security teams.
